# Elopeli Rautatie

Pregenerated / esikäsikirjoitettu Rautatie prototype for Elopeli.

This is a railroaded hidden-role / social deduction LARP demo where the AI generates the scenario structure and player-facing dramaturgical prompts before play. It is preserved as part of the Elopeli prototype lineage.

See [META_CONTEXT.md](META_CONTEXT.md) for the research context, findings, and lineage.

Original AI Studio app: https://ai.studio/apps/5f5fd124-c2d3-4329-8e04-9df6060c2bdb

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Copy `.env.example` to `.env.local` and set `GEMINI_API_KEY`.
3. Run the app:
   `npm run dev`
